《稀缺》

Posted on 2020-09-09 | Post modified: 2020-09-09 | In 所思所想

稀缺：我们是如何陷入贫穷与忙碌的

这个社会上的大部分人都是普通人，普通人无法得到丰富的资源，终究避免不了资源稀缺问题。其中，金钱稀缺和时间稀缺是两个最常见也是最可怕的稀缺。很好理解，想必你一定经历过资金短缺的情况，当时的你是不是非常痛苦，想到的解决办法可能就是借钱，先缓过这个时期，后面的事情以后再说；有时候工作压力较大，连续几周下来非常繁忙，几乎没有多少时间能够让你自由安排，接着你可能就错过了一些重要事情，比如陪家人、去检查身体等等，你的精力都用在了工作上而忽略了其他重要事情。这些处境只有在资源稀缺的情况下才能体会到，你被稀缺所困惑，眼前只有短期计划（利益），长期计划被忽略。

不难想象，只有短期计划的人生是怎么样的，看一下底层生活人民就懂了，一年忙到头也没赞多少钱，十几年下来生活也没有改善，依旧处于贫穷中。他们是如何陷入到其中，又是如何持续陷入无法走出的呢？这本书给了解答。

除此之外还有很多别的观点能帮助我们理解稀缺，明白稀缺是如何影响我们的行为和思想的，从而改变我们的生活。这个社会上的稀缺有很多，不止金钱和时间稀缺，还有社交稀缺、运动稀缺等等，只要我们明白了稀缺的本质以及影响，那么我们也不至于会一直陷入到稀缺中，然后忙碌一辈子依旧在稀缺中。

《随机漫步的傻瓜》

Posted on 2020-02-17 | Post modified: 2020-02-17 | In 所思所想

随机漫步的傻瓜

刚进入职场的头两年，我一直迫切的想要成功、想要走得更快，为了达成这个目的我看了很多书，也看了一些成功人士分享的经验，甚至为了得到更专业和真实的经验还参与过很多付费内容，最后将收获的这些“经验”进行整理、分析和付出实践。经过两年后也就是现在，我并没有取得非常大的成功，只能说是比以前过得要好，中途也放弃了很多当时看起来无比正确，可实际上却没什么用的经验。回想过去，可真是被收了很多智商税呢（笑。

并不是说经验无用，而是我没有明白理清逻辑，当时我犯了个错误，没有理解“幸存者偏差”这个逻辑谬误。多少在网上分享经验或者开设付费内容的人往往都是成功的，从他们的成功经验得出的结论是很单一的，不足以反映出真实情况。以及多数失败者不会选择在网络上发声，或者说不会被人关注，而成功者即使发声不多，也会被媒体大肆渲染。这一来一回，让真实信息更加远离现实，看到的不是根源而是表象。

最要命的是那些成功者也不一定是靠实力成功的，只是运气比较好罢了，可他们却不这样认为，把运气说成实力，将“随机事件”看待成“非随机事件”。而我们这些围观者将他们的经验汇集起来只能得到错误结论，得到一个错误结论，比没有结论更加可怕。

什么是“随机事件”和“非随机事件”？如何理解随机或者概率？那么就仔细阅读该书吧。

《拖延心理学》

Posted on 2020-01-28 | Post modified: 2020-01-28 | In 所思所想

拖延心理学

在生活中总是给自己定了很多目标，工作上制定了很多计划，可是最后没有一个是执行完成的，甚至几个月过去后都没有开始过。拖延一直伴随着我，只是有时候因为压力山大，被迫摆脱拖延，在某一段时间内超额完成计划，以为自己已经克服了拖延，成为自己想成为的人。当这段时间过去后，压力没有了，自己又变回原来的样子，明明特别想摆脱拖延，却一直陷于拖延。

我相信很多人都和我一样，特别想要提高时间效率，摆脱拖延带来的困扰，或者说扼杀拖延，让其不再出现。想要解决拖延这问题，就得先明白这问题是如何产生的，拖延因我们自己的心理而产生，了解自己的内心，这或许是“解决”拖延的最好办法。

想一下最近一个让你困扰的拖延问题，真正导致你迟迟不行动的原因是什么呢。害怕失败？害怕成功？害怕行动后的结果？这些都是心理问题，真正明白了这些原因，拖延的真相也就越来越近。

白天为生存，晚上谋发展

一天由二十四个小时组成，时间是有限的，我们白天会花费大量时间用于工作换取金钱为了生存，夜晚则会被睡眠占用大量时间，一天当中给留我们自己支配的时间已所剩无几，工作日是这样度过。而周末会有加班、应酬等情况，这也占用周末大量时间，还剩下一些时间也大概率是用来休息，缓解工作日带来的疲劳。感受一下，一周大部分时间都用在为了生存上，这样的一周为自己又干了什么呢。

当工作或者生存占用到个人大量时间后，很少有时间和精力来思考自己的问题（没错，这就是社畜），没有办法发展自己的兴趣、爱好和享受生活。想要跳出这个圈就必须得抽出属于自己的时间，工作日的晚上下班后到睡觉前这段时间，周末也能抽出一些。

为什么晚上谋发展对于个人来说那么重要？我自己感觉现在的公司无法给员工充分的依赖（不像以前在一家公司可以待很久），所以员工必须对自己的人生负责，要时刻关注自己。（否则，就像我一样随着工作年限增加，感受到时间流逝越来越快，除了工作上有所获得，自己本身好像没有得到些什么，也可谓“碌碌无为又一年”，逃。

Java 随想（一）

Posted on 2019-12-04 | Post modified: 2019-12-04 | In Java Framework

吾生也有涯，而知也无涯

吾生也有涯，而知也无涯。这句话大意是：我的生命是有限的，而知识是无限的。

知识是无限的，这句话反映在前几年出现的《得到》和《极客时间》APP中（在网络上传播的最广，拿来举例），里面充满了知识并且一直在更新，让人觉得不学习就落后，被时代抛弃了一样。但是呢，这些“学习知识”大都是无意义的，因为知识本来就是无限的，没必要掌握所有知识，被那些贩卖焦虑的人收智商税。只需要学习自己感兴趣或者对于工作来说必要的知识就行，知识都是有用的，区别在于如何理解和使用了。

现在对于学习某一个技术或者知识都有无尽的书籍，每本书籍的观点不完全相同，加上焦虑越来越多，能沉下心学习的人不多，我个人也是，也一样经常被焦虑困扰，每次都想系统性的学习，但是真的存在系统性的书籍么？这些书籍所讲述的内容一定是系统性的？值得思考。

现在记录的是自己平常对 Java 所思考的内容，每个人都有自己的理解，按照自己的理解去生活，这才是应该有的人生。「真的存在只要按照自己的理解生活的世界？雾，这是个哲学思考」每次学习了一部分知识之后再去看其他书籍，自己的思想又被影响了，不是说自己没有固定的立场或者无法独立思考，是真的存在某一个人的思想就完全是自己的，不受他人影响么。

好了，对于理性的思考就到这里吧，这是一个过于“复杂”的世界。接下来去一个过于“简单”的世界，里面只有冰冷的命令，它无法思考。

《自私的基因》

Posted on 2019-09-24 | Post modified: 2019-10-05 | In 所思所想

自私的基因

由于今年工作上的事情比较忙，所以看得书籍相对较少，其中，这本《自私的基因》算是今年看的书籍中对我影响最大的一本书。当看到书名中带有“自私”一词时，不知道大家心中作何感想。人本身由基因组成，既然基因是“自私”的，那么人本质上也应该是自私的，一切的活动都是为了繁殖，然后将基因遗传下去（不断复制）。

听起来有一点悲伤，我们人作为基因的载体，不管生活得怎么样，最后的目的是将基因遗传下去，我们（基因的载体）的一生都是为了这件事情而存在，“生命的意义”就在于此「达尔文主义」，如果真是这样的话，从我的角度来说是有些消极看待人生（这也是我现在想追求和思考的问题答案）。基因载体真的就无法摆脱基因的“控制”？不，其实人已经脱离了基因的控制，能证明这一点的就是避孕措施（载体作出的行为与基因的目的相违背）。

我读这本书的目的就是想了解人摆脱基因的控制到底是因为什么？如果基因没有遗传下去，那么人又是为什么要活下来，生命的意义从基因遗传又变成了什么？对于宇宙而言，“生命”很特别么？
